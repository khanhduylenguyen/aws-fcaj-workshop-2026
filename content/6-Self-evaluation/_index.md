During my **12-week internship in the AWS First Cloud Job (FCAJ) Workshop** from **01/04/2026** to **06/07/2026**, I had the opportunity to learn, practice, and apply my foundational cloud knowledge in a real working environment.

I participated in **building a full-stack web product on AWS** (including database design, Backend development with Spring Boot, Frontend with ReactJS, integration of AWS services such as S3, RDS, Bedrock, and CI/CD deployment via GitHub Actions), which significantly improved my skills in: **Java/Node.js programming, system design, Agile teamwork, technical reporting in both Vietnamese and English, and especially "cloud-native" thinking**.

In addition, I completed **3 in-depth technical blog posts** published on the workshop site, analyzing real-world AWS case studies (Blog 1 – processing unstructured PDFs with Textract, Blog 2 – digitizing medical records with Bedrock Data Automation + HealthLake, Blog 3 – contract extraction with Doczy.ai™). Writing the blogs helped me strengthen my research, information synthesis, and ability to communicate complex technical topics clearly.

### Needs Improvement

* **Deepen advanced cloud knowledge**: Although I am proficient with core AWS services (S3, Lambda, EC2, RDS, Bedrock, CloudWatch) at a basic-to-intermediate level, I still need to dig deeper into advanced topics such as **cost optimization with Cost Explorer & Savings Plans**, **multi-AZ/high-availability system design**, **advanced networking (VPC Peering, Transit Gateway, PrivateLink)**, **observability with X-Ray and CloudWatch Container Insights**, and **FinOps practices**. Goal: complete at least **2 AWS certifications** (SAA-C03 and one specialty such as Security Specialty or Solutions Architect Pro) within the next 6 months.
* **Strengthen deadline discipline**: Side-tasks (writing docs, updating backlog) were sometimes late because I was too focused on coding. I will improve by applying **time-blocking** techniques and using a **Jira dashboard** to track progress in real time.
* **Polish public-speaking skills**: I write well but still get a bit nervous presenting to large groups. I will attend more **lightning talks in the AWS User Group community** to practice.
* **Improve production debugging**: When systems run for real on AWS, tracing logs across CloudWatch, X-Ray, and application logs is still slow. I need to learn more about **distributed tracing** and **structured logging**.
